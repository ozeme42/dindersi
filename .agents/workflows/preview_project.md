# Projeyi Önizleme (Preview Project)

Bu proje Next.js tabanlıdır. Projeyi yerel ortamda çalıştırmak ve önizlemek için aşağıdaki adımları izleyin:

1. **Geliştirme Sunucusunu Başlatın:**
   ```bash
   npm run dev
   ```

2. **Önizleme:**
   Sunucu başarıyla başladıktan sonra (genellikle `localhost:3000`), önizleme bağlantısına tıklayarak tarayıcı üzerinden uygulamayı test edebilirsiniz.
