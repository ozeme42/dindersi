# Projeyi Derleme (Build Project)

Bu proje Next.js tabanlıdır. Projeyi sıfırdan kurmak ve derlemek için aşağıdaki adımları izleyin:

1. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   ```

2. **Tip Kontrolü (Typecheck):**
   ```bash
   npm run typecheck
   ```

3. **Üretime Hazırlama (Build):**
   ```bash
   npm run build
   ```
