# **App Name**: EduAI

## Core Features:

- Personalized Learning Path: AI generates a customized learning path based on the user's goals and skill level.
- Interactive Lessons: Create interactive lessons with videos, quizzes, and assignments to engage users.
- Progress Tracking: Monitor user progress and provide detailed performance reports.
- AI Tutoring Assistant: Offer 24/7 AI-powered assistance to answer student questions and provide hints using sophisticated reasoning tool.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke knowledge and trust.
- Background color: Very light gray (#F0F0F0).
- Accent color: Teal (#008080) for interactive elements and call to actions.
- Headline font: 'Space Grotesk', a sans-serif with a techy feel.
- Body font: 'Inter', a neutral sans-serif suitable for longer texts.
- Use modern and clean icons to represent various subjects and features.
- Subtle animations and transitions to enhance user engagement.
